В программе ships_battle.py реализована игра в морской бой с искусственным интеллектом(компьютером)
Правила такие же как в обычной игре "Морской бой". Есть два игрока, у каждого имеется стандартный набор кораблей: 
		один четырехпалубный, два трехпалубных, три двухпалубных, четыре однопалубных. Корабли расставляются на поле и игроки по очереди стреляют на поле врага размером 10*10 клеток.
		Игра заканчивается когда кто-либо из игроков потопит все корабли противника. 
Доступны следующие функции:
Ручная постановка кораблей на поле.
Рандомная генерация кораблей на поле.
Очистка поля.
Пропуск хода для игрока, с целью экстрима.
Ввод никнэйма.
Запись данных в SQL-таблицу(никнэйм, исход игры, время затраченное на игру и местная дата и время).
Просмотр всех результатов из SQL-таблицы.
